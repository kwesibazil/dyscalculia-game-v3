# How to setup environment variables

first, create an .env file in the root directory. 
Then supply the following env variable 

##Port
```
PORT
```
##Environment
 
```
NODE_ENV 
```

##Database Connection
###Production
```
PRODUCTION_DATABASE_URI 
```
###Development
```
DEVELOPMENT_DATABASE_URI 
```

##Secret
###Session Store
```
SESSION_STORE_SECRET 
```
###Session
```
SESSION_SECRET
```
